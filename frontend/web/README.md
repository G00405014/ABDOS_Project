# ABDOS Frontend

This is the frontend for the AI-Based Dermatological Observation System (ABDOS).

## Requirements

- Node.js v18+ (LTS recommended)
- npm v9+

## Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
